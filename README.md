# KayPyCode - 数据分析工具

一个基于 FastAPI 的数据分析 API 服务。

## 功能

- 基础数据统计分析
- CSV 数据处理
- 健康检查接口

## API 端点

- `GET /` - 欢迎页面
- `GET /health` - 健康检查
- `POST /analyze` - 数据分析接口
