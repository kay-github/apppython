# AGENTS.md

## 项目简介
本项目是一个 Hugging Face Docker Space（FastAPI），用于展示“新易盛 / 中际旭创 市值比例图”。页面为单一 Web 页面：运行时拉取最近一年的日线数据，并在启用实时时尽力补齐当天数据点，使用 Plotly 生成交互式图表。

## 主要功能
- 拉取最近一年的日线收盘价。
- 启用实时后，优先尝试追加当天实时行情（若获取到任一只股票价格，也会补一条当天记录）。
- 计算市值比例与折扣（比例 * 10）。
- 返回单页 HTML（带 Plotly 交互图）。
- 页面显示数据日期、实时状态、数据来源。
- 图表支持“近1周/近1月/近1季/近半年/近1年”区间按钮与周期统计卡片（最新/最高/最低）。
- 首页有 20 秒短缓存（可通过 `?refresh=1` 强制刷新）。

## 关键文件
- `app.py`：FastAPI 服务与图表生成逻辑。
- `requirements.txt`：依赖列表（FastAPI、Plotly、Requests 等）。
- `Dockerfile`：HF Docker Space 启动入口（`uvicorn app:app`）。
- `README.md`：Space 配置与项目说明（含 YAML 头）。

## 数据源（当前免费方案）
- **日线数据**：
  1) 东方财富 K 线（日线）
  2) 失败则切换 新浪日线
- **实时数据**：
  1) 东方财富实时
  2) 失败则切换 新浪实时
  3) 再失败切换 腾讯实时

## 关键参数（在 app.py 顶部）
- `ZJXC_CODE` / `XYS_CODE`：股票代码（支持 `sz.300308` / `300308.SZ`）。
- `ZJXC_MARKET_CAP` / `XYS_MARKET_CAP`：市值锚点（单位：亿元），用于反推股本。
- `ZJXC_SHARES` / `XYS_SHARES`：显式股本（单位：亿股）。**仅当两者都非 `None` 时**才会覆盖市值反推。
- `USE_REALTIME`：是否尝试实时行情。
- `FETCH_RETRY` / `RETRY_DELAY_SECONDS`：日线数据重试配置。
- `SINA_DATALEN`：新浪日线接口拉取条数上限。
- `CACHE_TTL_SECONDS`：页面缓存秒数（默认 20 秒）。

## 计算逻辑
1. 拉取两只股票最近一年日线收盘价（东财优先，失败走新浪）。
2. 若启用实时，则尝试获取两只股票实时价；任一成功时，会写入当天数据点（另一只失败则沿用其最新日线收盘价）。
3. 通过市值锚点或显式股本推算日度市值（显式股本需两只都配置才生效）。
4. 计算市值比例与折扣：
   - `ratio = xys_mv / zjxc_mv`
   - `discount = ratio * 10`
5. 生成 Plotly 图表（均值线、最高/最低点、区间按钮），返回页面。

## 页面说明
- 标题：新易盛 / 中际旭创 市值比例图
- 顶部信息：数据日期、是否使用实时、生成时间、日线来源、实时来源
- 右上角按钮：刷新获取最新数据
- 中部“周期统计（固定）”卡片：随图表区间按钮切换显示该区间的最新/最高/最低折扣及日期

## 部署要点（HF Space）
- Space 类型：Docker
- 端口：`7860`
- `README.md` 顶部必须有正确的 YAML 头（`sdk: docker` / `app_port: 7860`），否则会报“Configuration error”。

## 常见问题
- **使用实时行情显示“否/部分”**：说明实时接口取值失败或仅一只股票成功。
- **日线失败**：可能是上游接口阻断或波动，检查网络或更换来源。
- **数据不一致**：市值锚点日期与最新行情不一致会导致偏差。
- **刚刷新没变化**：可能命中 20 秒缓存；使用右上角按钮（`?refresh=1`）可强制重算。

## 修改建议
- 想改股票：只需改 `ZJXC_CODE` / `XYS_CODE`。
- 想改口径：改 `ratio` 计算或直接拉取真实市值（如能获取）。
- 想调整样式：改 HTML/CSS 或 Plotly 配色区块。
